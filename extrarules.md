# Extra Rules

## Hay Roll Multiplier

When calculating **Hay** roll payouts in the roll table:

- If Hay quantity is **5–9**, apply a **1.5×** multiplier.
- If Hay quantity is **10+**, apply a **2×** multiplier.
- Otherwise, apply **1×**.

This multiplier affects the **roll payout values** shown in the roll table (profit per roll), not the asset purchase net values.
